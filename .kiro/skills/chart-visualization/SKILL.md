# Chart Visualization

When presenting EEMO reporting and analytics data, select the most appropriate visualization based on the information being displayed.

## Recommended Chart Types

* Revenue Trends → Line Chart
* Revenue by Facility → Bar Chart
* Collection Performance by Period → Line Chart or Bar Chart
* Paid vs Unpaid Vendors → Pie Chart or Donut Chart
* Delinquency by Facility → Bar Chart
* Monthly Collection Status → Stacked Bar Chart
* Contract Expiration Monitoring → Table with KPI Cards
* Daily Collection Activity → Calendar Heatmap
* Terminal Trip Collections → Line Chart or Data Table
* Slaughterhouse Transactions by Animal Type → Bar Chart
* Outstanding Balances by Facility → Bar Chart

## Visualization Principles

* Select the simplest chart that communicates the information clearly.
* Prioritize readability and accuracy over visual complexity.
* Use tables when exact values are important.
* Use KPI cards for high-level summary metrics.
* Avoid radar charts, Sankey diagrams, network graphs, word clouds, and other complex visualizations unless specifically requested.
* Ensure charts are suitable for government reporting and administrative decision-making.
